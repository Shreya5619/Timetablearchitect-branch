API Documentation

A guide to using the functions from teachers.ts and room.ts.

Teachers Module

createTeachers

Creates a new teacher in the database.

  • Parameters: JWT token, name, initials, email, department, alternate departments, timetable, labtable
  • Returns: Status code and the created teacher object
  • Example: 
    createTeachers(JWTtoken, "John Doe", "JD", "john@example.com", "EI","CSE,AIML");

getTeachers

Retrieves a list of teachers based on user role and department.

  • Parameters: JWT token
  • Returns: Status code and a list of teachers
    • getTeachers("JWTtoken");

peekTeacher

Fetches details of a specific teacher.

  • Parameters: Token, name, department
  • Returns: Status code and the teacher object
    • peekTeacher("JWTtoken","Kendaganna","CSE");

Room Module

createRoom

Creates a new room in the database.

  • Parameters: JWT token, name, lab (boolean), timetable, department
  • Returns: Status code and the created room object
    • createRoom(JWTtoken, "Physics Lab", true);

getRooms

Retrieves a list of rooms based on user role and department.

  • Parameters: Token
  • Returns: Status code and a list of rooms
    • getRooms(JWTtoken);

peekRoom

Fetches details of a specific room.

  • Parameters: Token, name, department
  • Returns: Status code and the room object
    • peekRoom(JWTtoken, "Physics Lab");
